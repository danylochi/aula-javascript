# Aula JavaScript

## Sobre o projeto
Projeto introdutorio de JavaScript para praticar conceitos basicos em um unico arquivo.

O foco e exercitar arrays, manipulacao de strings, variaveis e saida no console.

## Estrutura do projeto
- `main.js`: arquivo principal com exemplos de estudo (parte ativa e parte comentada).

## Pre-requisitos
- Node.js instalado (LTS recomendada).

## Como executar
No terminal, dentro da pasta do projeto:

```bash
node main.js
```

Saida esperada:
- Lista de frutas no console.
- Quantidade de itens da lista.

Exemplo:

```text
Lista: [ 'maca', 'pera', 'laranja' ]
Quantidade de itens: 3
```

## O que este codigo demonstra
- Criacao e uso de array (`lista`).
- Operacoes em array com `push` e `pop` (comentadas no arquivo).
- Uso de `console.log`.
- Declaracao de variaveis com `var`.
- Operacoes com numeros e strings em exemplos comentados.
- Metodos de string como `toLowerCase()` e `replace()` nos trechos comentados.
